# Эталонная структура силлабуса

**НАО «Медицинский университет Астана»**

Стандарт приведен к шаблону ДАР от **12.06.2026**:
`syllabus-template от ДАР 12.06.26.docx`.
Язык заполнения — русский.

---

## Общие правила

Силлабус может быть двух типов:

| Тип | Код | Как используется |
|---|---|---|
| Модульный | `module` | Один документ для модуля; разделы по дисциплинам повторяются там, где это требуется шаблоном |
| Дисциплинарный | `discipline` | Один документ для одной дисциплины |

Источники данных:

| Тег | Источник | Смысл |
|---|---|---|
| 🟦 | **Platonus** | Данные из БД Platonus / РУП |
| 🟨 | **Отдел / библиотека** | Справочники, согласование, карта обеспеченности |
| 🟩 | **ППС** | Заполняет составитель силлабуса |
| ⬜ | **Вычисляется** | Автоматически из других полей |

| СРО | 🟩 ППС |

В шаблоне ДАР от 12.06.2026 тематический план может иметь двухстрочную шапку:
верхний уровень «Количество учебных часов», нижний уровень `Л / ПЗ / СРОП / СРО`.
При проверке суммы по каждому виду часов (`Л`, `ПЗ`, `СРОП`, `СРО`) должны отдельно
совпадать с разделом 1 «Описание дисциплины (модуля)».

## 8. Политика дисциплины (модуля)

Политика общая для документа, заполняется один раз. Шаблон ДАР содержит типовые формулировки по следующим блокам:

| Блок | Содержание |
|---|---|
| Общие обязанности | Выполнение образовательной программы и силлабуса, посещение занятий, своевременная сдача работ |
| Посещаемость | Обязательное посещение, учет в АИС Platonus, опоздание более 15 минут как отсутствие |
| Отработка пропусков | Уважительные причины отрабатываются по Академической политике; неуважительные пропуски занятий и ТК не отрабатываются |
| Снижение оценки | Пропуск лекции: 1 балл; пропуск практического, лабораторного или симуляционного занятия: 10-20 баллов |
| Допуск к промежуточной аттестации | ОРД не менее 50 баллов и выполнение обязательных видов учебной работы |
| Дресс-код и клинические базы | Внешний вид, санитарная книжка, флюорография, медицинская форма и иное оснащение |
| Электронные устройства | Только в образовательных целях или с разрешения преподавателя; на контроле и аттестации запрещены |
| Академическая честность | Соблюдение Кодекса академической честности и Академической политики |
| Апелляция | По техническим причинам в течение 24 часов после публикации результатов |
| Доступность | Равные условия обучения и оценивания, инклюзия, недискриминация, гендерное равенство, устойчивое развитие |

## 9. Оценивание

### 9.1. Структура оценивания

| Компонент | Вес | Срок | Критерии |
|---|---:|---|---|
| Текущий контроль успеваемости | 60% | Согласно тематическому плану | Согласно чек-листам текущего контроля |
| Промежуточная аттестация | 40% | В период экзаменационной сессии | Согласно критериям промежуточной аттестации |

Формулы:

- `ОРД` = среднеарифметическое значение результатов текущего контроля.
- `ОПА` = оценка промежуточной аттестации.
- `ИО = ОРД * 0,6 + ОПА * 0,4`.
- К промежуточной аттестации допускаются обучающиеся с рейтингом допуска не менее 50 баллов.

### Текущий контроль

В шаблоне ДАР предусмотрены отдельные формы текущего контроля:

| Подраздел | Форма | Структура |
|---|---|---|
| 9.2 | Устный экзамен | Чек-лист: №, критерий, макс. балл, балл 0-10 |
| 9.3 | Письменный экзамен | Чек-лист: №, критерий, макс. балл, балл 0-10 |
| 9.4 | Тестирование | №, количество вопросов, уровень сложности, балл за задание, баллы итого |
| 9.5 | Ситуационная задача | №, критерий, макс. балл, балл 0-10, комментарий |
| 9.6 | OSCE-станция | Навык ПН, название станции, баллы; далее алгоритм станции и чек-лист |

## Правила для модульного и дисциплинарного силлабуса

- `discipline`: титул, описание, РО, ПН, тематический план, оценивание, литература, КИС и КО относятся к одной дисциплине.
- `module`: титул и цель/краткое содержание относятся к модулю; описание, РО, ПН, тематический план, оценивание, литература, КИС и КО заполняются по каждой дисциплине, когда их содержание различается.
- Политика, глоссарий, график СРОП и согласование обычно заполняются один раз для всего документа.
