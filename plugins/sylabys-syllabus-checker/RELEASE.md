# Release 1.0.2

## Что изменилось

- Обновлена инструкция для ППС: установка идёт через Codex app и Git marketplace, без Share-сценария и без CLI-команд для преподавателя.
- Пересобрана PDF-инструкция со всеми реальными скриншотами интерфейса Codex app.
- Уточнено, что Git for Windows нужен как системная зависимость для добавления GitHub marketplace из Codex app.
- Экстрактор переписан на table-first и только стандартную библиотеку (без python-docx/openpyxl): корректно извлекаются часы, преподаватели, тематический план, глоссарий, структура оценивания, согласование.
- Адресная сверка с ОП и РУП: берутся только данные нужной дисциплины/модуля. РУП-сверка учитывает «модуль = сумма дисциплин» и сама выбирает охват (одна дисциплина или агрегат модуля).
- Сверка с ОП сделана scope-aware (модуль vs дисциплина) — без ложных FAIL по кредитам и матрице РО.
- Заглушки заменены настоящими проверками: КИС (INT-046), литература (INT-050), глоссарий (INT-051), график СРОП (INT-052), титул vs раздел 1 (INT-056), диагностичность РО (TXT-004).
- Поддержан вариативный шаблон силлабуса: комбинированная таблица РО/ПН/тематический план с двухстрочной шапкой часов + синтез РО/ПН из плана.
- Громкие ошибки вместо тихой пустой экстракции; убран доменный хардкод из эвристики TXT-001.

## Проверка

- Прогон на трёх реальных силлабусах (анатомия, патология, кардио-респираторный) — все FAIL подтверждены как настоящие замечания.
- 8 модульных тестов зелёные (`python -m unittest tests.test_det_checks`).
