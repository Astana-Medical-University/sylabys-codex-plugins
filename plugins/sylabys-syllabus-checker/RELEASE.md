# Release 0.1.8

## Что изменилось

- Исправлен парсинг курса: значение больше не берётся из карты обеспеченности литературы и строки `Всего контингент обучающихся 735`.
- Курс теперь извлекается строго из паспортного блока / периода обучения и ограничивается допустимым диапазоном 1-7.
- Для модульного силлабуса кредиты в финальном отчёте больше не подаются как рассчитанный объём модуля без сверки с РУП.
- В отчёт добавлена оговорка: найденные кредиты модуля являются паспортным значением из силлабуса, а итоговый объём нужно подтверждать по РУП.
- Добавлены регрессионные тесты на ошибку `course=735` и вывод `735, 2.0 кредитов`.
